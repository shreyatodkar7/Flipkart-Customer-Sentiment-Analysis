 🛒 Flipkart Customer Sentiment Analysis

 📖.Overview 
The Flipkart Customer Sentiment Analysis project is a Machine Learning-based web application developed to analyze customer reviews and determine whether the sentiment expressed is Positive or Negative. With the rapid growth of e-commerce platforms, customers rely heavily on online reviews before making purchasing decisions. This project aims to automate the process of understanding customer opinions by using Natural Language Processing (NLP) and Machine Learning techniques.
The application processes textual customer reviews by performing text preprocessing, feature extraction using the TF-IDF Vectorizer, and sentiment prediction using a trained Machine Learning model. A simple and interactive web interface is developed using Flask, allowing users to enter a product review and instantly receive the predicted sentiment.
The project is built using Python and various data science libraries such as Pandas, NumPy, Scikit-learn, and NLTK. It demonstrates the complete machine learning workflow, including data preprocessing, model training, evaluation, and deployment. This project provides valuable insights into customer feedback and can help businesses understand customer satisfaction, improve products, and make data-driven decisions.

✨.Features
• Predicts Positive and Negative sentiments.
• User-friendly Flask web interface.
